# Raspi-Car

## 简介
树莓派远程遥控车，树莓派作为小车的主控，负责与控制端连接。

1、接收来自控制端的指令并执行，如小车的前后左右控制、开关灯和摄像头开关等操作。

2、 采集车载摄像头信息，并回传至控制端。
## 起因
最近入手了一块树莓派4B，打算搞一辆远程控制小车，顺便学习的视频获取、传输相关的知识。

## 功能
- 视频回传+控制
- 手机端+PC都能操控
- frp+4G网络，远程控制

## 相关知识
- 视频读取 opencv
- 视频传输 socket
- pwm输出 GPIO

## 运行环境
请安装python 3.6 运行此项目
- [Python 3.6](https://www.python.org/)
